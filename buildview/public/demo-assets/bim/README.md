# BIM / IFC samples (links, not bundled)

IFC files can be large and some have per-file licenses, so we link curated,
clearly-usable sources here instead of committing binaries. Add a specific file
only after confirming its exact license, and record it in `../ASSETS_CREDITS.md`.

| Source | License | Notes |
|---|---|---|
| [buildingSMART Community Sample Test Files](https://github.com/buildingsmart-community/Community-Sample-Test-Files) | CC BY 4.0 | Community openBIM samples (IFC/BCF/IDS). Attribute. |
| [buildingSMART official Sample-Test-Files](https://github.com/buildingSMART/Sample-Test-Files) | verify per repo | Schema-version IFC samples. |
| [Duplex Apartment — WBDG/NIBS "Common BIM Files"](https://www.wbdg.org/bim/cobie/common-bim-files) | NIBS public test files | Residential BIM (matches our theme); confirm re-host terms. |
| [IFC Wiki — Examples (e.g. FZK-Haus)](https://www.ifcwiki.org/index.php/Examples) | per-file | Small single-family house model. |
| [OpenIFC Model Repository (Univ. Auckland)](https://openifcmodel.cs.auckland.ac.nz/) | per-model | Curated IFC test models. |
